Cacharreando

Acostumbro a cacharrear y aprender nuevas tecnologías y productos de desarrollo de software mediante tutoriales, videos de youtube, cursos o meetups. La mayoría de las veces implementando simples PoCs que no llevan a ningún resultado tangible pero con alguna aplicación completa terminada ocasionalmente. Algunos ejemplos:

  • Desde bien pequeño hice mis primeros "pinitos" en la programación haciendo aplicaciones con Visual Basic y páginas web como una (ya cerrada) para mi pequeño pueblo Martínez (Ávila).
  • Durante bachillerato antes de comenzar la carrera seguía mejorando mis capacidades de desarrollo de software llegando a crear un troyano con c++ (con fines académicos por supuesto).
  • Durante la carrera implementé varias aplicaciones para dispositivos móviles. Por ejemplo una app para jugar al “Mus” en JME y aplicaciones Android nativas “MathBall” y “FastFinger”.
  • Cuando los microservicios empezaron a ponerse de moda implementé un PoC de arquitectura de microservicios con Spring Cloud.
  • Diseñé y construí esta pagina web personal durante la cuarentena por Coronavirus.
